# 🚀 Premium M3U8 Player

A state-of-the-art, high-performance web-based M3U8 player built with **HLS.js** and a modern **Glassmorphism UI**. Designed for speed, flexibility, and a premium viewing experience.

---

## ✨ Key Features

- 🎥 **Advanced HLS Playback**: Seamless streaming using the latest `hls.js` library.
- ⚙️ **Quality Selector**: Manually switch between 1080p, 720p, 480p, or use **Auto** mode.
- 🎨 **Glassmorphism UI**: A stunning, modern interface with smooth animations and blur effects.
- 📱 **Mobile Optimized**: Full touch support with **Swipe Gestures** (Swipe for Volume & Seeking).
- ⌨️ **Keyboard Shortcuts**: Full control via keyboard (Space, F, M, J/K/L, and Arrow Keys).
- 🚀 **Fast Loading**: Optimized buffer management for low-latency streaming.
- 🔘 **Big Play Button**: Reliable autoplay recovery for modern browsers.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism), JavaScript (ES6+)
- **Libraries**: 
  - [HLS.js](https://github.com/video-dev/hls.js/) (Streaming Engine)
  - [jQuery](https://jquery.com/) (DOM Manipulation)
  - [Mousetrap](https://craig.is/killing.it/mousetrap) (Keyboard Shortcuts)
  - [Font Awesome](https://fontawesome.com/) (Premium Icons)

---

## 🚀 How to Use

1. **Open `index.html`** in your browser.
2. **Paste your .m3u8 link** into the beautiful input field.
3. Click **Launch Player** and enjoy the premium experience!

### Keyboard Shortcuts
| Key | Action |
|-----|--------|
| `Space` / `K` | Play / Pause |
| `F` | Toggle Fullscreen |
| `M` | Mute / Unmute |
| `Right` / `Left` | Seek 5s |
| `J` / `L` | Seek 10s |
| `Up` / `Down` | Volume Control |
